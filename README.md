# miracle
当前工程包含的Magic：
- `@what-a-faka/utils`，一系列通用工具包
- `@what-a-faka/vue-authplugin`，Vue权限插件

## 使用方法
### 安装
```
$ npm i @what-a-faka/utils
```

### 使用
```
import { mosaicFormat } from '@what-a-faka/utils'

mosaicFormat.mosaicPhone('13900000000') // 1390****000
```
